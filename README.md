# DragonflySvg

Dragonfly analyser and processor for SVGs.

Uses the [nokogiri](http://nokogiri.org) gem for SVG parsing.

    set_view_box
    set_dimensions

## Installation

Add this line to your application's Gemfile:

    gem 'dragonfly_svg'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dragonfly_svg
